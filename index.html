INSTALLATION GUIDE

(After Kez patch, there is a new method to install mods)

Step 1: Set language option in game 'settings' to kr/ ru/ cn etc (not En)

Step 2: Drop pak file(s) from 'VPK folder' into 'dota_language' folders under 'dota 2 beta/game' directory.

Step 3: Start Dota
